# ObsessiveWP Splash Page

This repository contains a single-page static site for the ObsessiveWP closing splash page.

## Local preview

Run a static file server from the repository root:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deployment

This site can be deployed directly to:

- GitHub Pages
- Netlify
- Cloudflare Pages

No build step is required.
